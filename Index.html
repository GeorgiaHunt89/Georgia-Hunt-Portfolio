Previous work

Here are some examples of my previous applications that are available to view and trial through GitHub and Heroku. Simply click on the slide to view the GitHub page.


Recipe Scraper

A simple application that allows the user to input a recipe and receive all the recipe data scraped and saved to a recipe card within their own dashboard.
The motivation behind the recipe scraper was to offer an easy way to track and view various recipes from different sites.
This application employs the use of NPM packages including recipe scraper, express session, express handlebars, Bcrypt and Dotenv while also implementing frameworks and modules from Bulma, Sequelize ORM, ExpressJS and MySQL2.

Created 25th June 2021

Bitcoin Bubbler

A application that allows individuals to monitor the current Bitcoin price, monthly change in US dollars and %. The user has the option to input an amount of bitcoin and the price in a range of different currencies.
The motivation behind the Bitcoin Bubbler was to offer crypto currency holders an easy way to track and view the price of their bitcoin.
This application employs the use of API packages CoinDesk and CurrencyConverter.

Created 20th April 2021

Team Profile Generator

A Node.js command-line application that allows a user to input information about team members, then generates an HTML webpage that displays summaries for each person.

Created 8th June 2021
